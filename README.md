# SEM-ISUC-2019

### Español

El presente repositorio contiene el archivo Rmarkdown que se utilizó para el análisis de datos en el proyecto "Preferencias por políticas sociales: Evaluación desde un modelo de ecuaciones estructurales para el caso chileno", por José Daniel Conejetos (GitHub: JDConejeros) y Pío Marshall. El proyecto se desarrolló en el contexto del curso de Modelo de Ecuaciones Estructurales, en el Instituto de Sociolgía de la Pontificia Universidad Católica de Chile el año 2019.

El objetivo del proyecto era analizar la relación entre auto-interés, medido a través del quintil de ingreso, y preferencia por políticas sociales redistributivas a través de modelos de ecuaciones estructurales.Para esto se realizó (i) un análisis confirmatorio de dos dimensiones de políticas redistributivas y (ii) un análisis de invarianza según quintil de ingreso.

### English

This repository contains the Rmarkdown file used in the data analysis from the project "Preferences for social policies: Evaluation from a model of structural structures for the Chilean case", by José Daniel Conejetos (GitHub: JDConejeros) and Pío Marshall. The project was developed in the context of the Structural Equations Model course, at the Institute of Sociology of the Pontifical Catholic University of Chile in 2019.

The objective of the project was to analyze the relationship between self-interest, measured through the income quintile, and preference for redistributive social policies through structural equation models. For this, (i) a confirmatory analysis of two dimensions of redistributive policies was carried out and (ii) an analysis of invariance according to income quintile.
