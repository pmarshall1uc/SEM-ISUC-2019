# SEM-ISUC-2019

### Español

El presente repositorio contiene el archivo Rmarkdown que se utilizó para el análisis de datos en el proyecto "Preferencias por políticas sociales: Evaluación desde un modelo de ecuaciones estructurales para el caso chileno", el cual se desarrolló en el contexto del curso de Modelo de Ecuaciones Estructurales, en el Magíster de Sociología de la Pontificia Universidad Católica de Chile el año 2019.

El objetivo del proyecto era analizar la relación entre auto-interés, medido a través del quintil de ingreso, y preferencia por políticas sociales a través de modelos de ecuaciones estructurales.Para esto se realizaró (i) un análisis confirmatorio de dos dimensiones de políticas redistributivas y (ii) un análisis de invarianza según quintil de ingreso.
